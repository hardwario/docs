# EMBER Changelog

This page tracks all notable changes across the EMBER platform — including **firmware** and **hardware**. Use the tabs below to filter by change category.

:::info

EMBER is based on **MikroTik RBM33G** hardware running **RouterOS**. Firmware updates are managed through MikroTik's own update mechanism (RouterOS auto-update via WebFig or CLI). There is no separate HARDWARIO-published firmware repository for EMBER.

For RouterOS release notes, refer to [MikroTik Changelog](https://mikrotik.com/download/changelogs).

:::

# External Sensors

GLIDER accepts several types of external inputs in addition to its internal sensors:

- **DS18B20 temperature probes** on the 1-Wire bus - see [**Temperature Sensors**](temperature.md).
- **Digital inputs** CH1 / CH2 - see [**Configuration → Digital Inputs**](../configuration.md#digital-inputs).
- **Modbus RTU** devices on the isolated RS-485 port - see [**Shell Commands**](../commands/shell-commands.md).

Pick the relevant page below to see wiring, configuration and example commands.
